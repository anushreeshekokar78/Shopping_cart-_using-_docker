# Mini Shopping Cart API

A lightweight Dockerized shopping cart microservice built with Python Flask.

## Features

- 🛍 Product listing
- 🛒 Add/remove items from cart
- 💰 Automatic total calculation
- 🐳 Docker containerized
- 📦 In-memory storage
